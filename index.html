Technologies

HTML5

HTML5 constitue la base de mes développements web. Il me permet de structurer efficacement les pages avec des balises sémantiques modernes, garantissant accessibilité, performance et compatibilité sur tous les navigateurs.

CSS3

Grâce à CSS3, je conçois des interfaces web esthétiques, responsives et animées. J’exploite les dernières fonctionnalités comme les flexbox, grid, transitions et animations pour offrir une expérience utilisateur fluide et agréable.

JavaScript

JavaScript me permet de rendre les interfaces interactives et dynamiques. Je l’utilise pour gérer les événements, manipuler le DOM, et développer des fonctionnalités avancées côté client comme les effets dynamiques, les validations et les requêtes API.

GitHub

Git et GitHub sont au cœur de ma méthodologie de travail. Ils me permettent de collaborer efficacement sur des projets, suivre les évolutions du code, créer des branches de développement et déployer des solutions fiables.

WordPress

J’utilise WordPress pour concevoir des sites administrables rapidement et efficacement. Je maîtrise l'installation, la personnalisation de thèmes, l’intégration de plugins et l’optimisation SEO pour des sites professionnels.

PHP

PHP reste une technologie incontournable pour concevoir des applications robustes et scalables. Je l’ai appliqué dans divers projets allant de simples sites vitrines à des systèmes complexes, en procédural comme en orienté objet.

Laravel

Laravel est mon framework PHP de prédilection pour le back-end. Grâce à MVC, au routage clair et à Eloquent, je conçois des applications robustes, sécurisées et maintenables.

Node.js

Avec Node.js, je développe des API REST rapides et évolutives, avec logique métier, sécurité, connexion MongoDB/MySQL, etc.

Angular

Angular est mon framework principal pour le front-end. Ses composants, routing et services injectables permettent de créer des interfaces dynamiques, connectées aux API.
